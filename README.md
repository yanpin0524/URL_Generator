## 產品功能

1. 輸入網址轉換成短網址
2. 點擊Copy按鈕複製網址
3. 透過短網址轉址到原本網址

## 專案安裝流程

1. 請先安裝 Node.js、npm 與 Mongodb
2. 打開你的終端機，開啟資料庫，並將此專案 Clone 到本地
```
git clone https://github.com/yanpin0524/URL_Generator
```
3. 之後進入專案資料夾，輸入
```
npm install
```
4. 安裝完畢後，輸入以下內容 開始運行網站
```
npm run start
```
5. 若是跑出 "伺服器已連線 : localhost:3000" 代表成功，現在可以輸入網址，並開始使用了
```
http:/localhost:3000
```
6. 要結束運行請按下 ctrl + c
